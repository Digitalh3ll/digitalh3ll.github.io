[1] Introdução

Sabe quando você vai ao supermercado e vê aqueles produtos com uma etiqueta pequena colada neles,
ou quando você usa um transporte publico e paga com a sua identificação de estudante?
Bem, essas cartões fazem parte de um sistema chamado RFID.

[2] Oque é RFID?

RFID significa Identificação por Radiofrequência (em inglês, Radio Frequency Identification). É uma tecnologia que permite a identificação e o rastreamento de objetos por meio de sinais de radiofrequência.
Ao contrário dos códigos de barras tradicionais, o RFID não requer contato direto para a leitura e pode capturar informações de forma rápida e eficiente.
O sistema RFID é composto por três Elementos principais:

- Leitor e um software de gerenciamento de dados. A tag RFID contém um chip eletrônico e uma antena, responsáveis por armazenar e transmitir informações através das ondas de rádio.
Essas tags podem ser de diferentes tipos, como tags passivas (que não possuem bateria própria e são ativadas pelo campo eletromagnético do leitor) ou tags ativas (que possuem bateria própria e podem transmitir informações)

- O leitor RFID é o dispositivo responsável por emitir ondas de rádio e captar os sinais enviados pelas tags. Ele também pode ser conectado a um computador ou a um sistema central para processar as informações coletadas.
O software de gerenciamento de dados permite armazenar, organizar e analisar as informações coletadas pelas tags RFID.

[3] Como funciona? (breve explicação)

O cartão é equipado com uma etiqueta especial chamada tag RFID, Essa tag contém um pequeno chip e uma antena.
Quando um leitor RFID próximo à tag emite um sinal de rádio, a antena da tag captura essa energia, Com a energia recebida
o chip da tag é ativado e transmite as informações armazenadas nele de volta para o leitor.
O leitor recebe essas informações e pode usá-las para identificar o objeto, como um número de série, por exemplo.

#include 
#include 

#define RST_PIN         9    
#define SS_PIN          10     

MFRC522 mfrc522(SS_PIN, RST_PIN);  // Cria A instancia Do MFRC522

void setup() {
	Serial.begin(9600);		// Inicia a comunicação com o computador
	while (!Serial);		// Não faça nada se nenhuma porta serial for aberta (adicionado para Arduinos baseados em ATMEGA32U4)
	SPI.begin();			// Barramento SPI de inicialização
	mfrc522.PCD_Init();		// Inicialização MFRC522
	delay(4);				// Dalay Opcional
	mfrc522.PCD_DumpVersionToSerial();	// Mostrar detalhes de PCD - MFRC522 Card Reader detalhes
	Serial.println(F("Examine o PICC para ver os blocos de UID, SAK, tipo e dados..."));
}

void loop() {
	// Redefina o loop se nenhuma nova placa estiver presente no sensor/leitor. Isso economiza todo o processo quando ocioso
	if ( ! mfrc522.PICC_IsNewCardPresent()) {
		return;
	}

	// Selecione um dos cartões
	if ( ! mfrc522.PICC_ReadCardSerial()) {
		return;
	}

	// Despejar informações de depuração sobre o cartão; PICC_HaltA() é chamado automaticamente
	mfrc522.PICC_DumpToSerial(&(mfrc522.uid));
}
